# AI Resume Analyzer

A full-stack AI Resume Analyzer project using FastAPI and React.

## Features
- Upload PDF resumes
- Extract text
- Analyze skills
- ATS-style scoring
- Missing keyword suggestions
- REST API backend

## Tech Stack
- Frontend: React
- Backend: FastAPI
- AI: OpenAI API
- Database: SQLite

## Run Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

## Run Frontend

```bash
cd frontend
npm install
npm start
```
